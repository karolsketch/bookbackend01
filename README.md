# bookbackend01
📚 Online Bookstore Backend — Spring Boot + MySQL  
書店購物車後端系統（含會員、訂單、付款方式、Email 驗證）
本專案為書店電商系統之後端，採用 Spring Boot 3 + JPA + MySQL 開發，符合三層式架構（Controller / Service / Repository）。
支援會員註冊登入、書籍查詢、付款方式、建立訂單、查詢訂單。
內容完整、結構清楚、適合畢業專題與面試作品。
